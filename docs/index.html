HanX Blockly 是一个基于 Tauri 和 Blockly 的现代化 IDE，专为 Python 和 Arduino 开发者设计。具备智能硬件发现、插件系统与原生性能体验。

核心特性

🔌

智能硬件发现

自动扫描并识别 Arduino 开发板及其串口，真正做到即插即用。

🧩

强大的扩展性

通过简单的 JSON 和 JS 定义，即可动态导入第三方组件库和新积木。

🚀

原生性能

基于 Rust (Tauri) 开发，秒级启动，丝滑流畅的交互体验。

📟

实时监控

内置高可靠串口终端，调试硬件输出从未如此简单。
